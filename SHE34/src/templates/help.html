How to use SHE? {# How to use SHE#}




About SHE


SHE or Smart Heurisic Evaluation is a collaborative web application for supporting Heuristic Evaluation. The process of systematic development of user friendly interfaces or usability engineering involves constant evaluation of a system design through usability tests and inspection methods. Heuristic evaluation (HE) is one of the most important inspection methods in usability engineering, in which a number of usability experts evaluate the user interfaces based on a set of pre-defined principles or the so called "heuristics" and at the end, their findings will be merged in a summary report.

SHE is tool for supporting the whole process of heuristic evaluation. It is smart enough to recognize similarities between evaluations and recommend them as a potential merging candidate in the final report.

How to Be an Evaluator in SHE?


Users will see in their dashboard whether or not they are an evaluator in a project of other users:



If the deadline of a project is not passed an evaluator can see the blue button Evaluate. By clicking this button an evaluator can start adding his/her evaluations to the project. Evaluators can remove themselves as an evaluator in a specific project by pressing the Del button. The delete process will be finished after the confirmation in a pop up warning window.



Adding a New Evaluation


If the project deadline for the evaluation is not passed an evaluator can press the add evaluation button to open an evaluation form similar to the following screenshot. By filling the form and pressing the submit button at the bottom of the page the evaluation will be saved. In this form some fields are optional and some are necessary. Title, place and description fields are required to be filled. The rest is optional but recommended; e.g. the link field in the evaluation of a website page which should be filled in this form: http://www.example.com.



Adding a New Environment


Evaluators can register the specifics of their environment so the manager can see the circumstances of the evaluations e.g. monitor resolution, operation system, etc. Creating an environment item is done by clicking on the link create an evaluation environment which is in the sidebar of the dashboard and will opens a form similar to the following screenshot. An evaluator can create multiple environments.



How to Be a Manager in SHE?


Being a manager means to define and edit a project. A manager can add and delete other users as evaluators of the project and finally can merge the evaluations and create reports of the results of the evaluation. A manager can define and edit heuristic sets upon which a project is defined.



Defining a Heuristic Set


SHE already has some widely used set of heuristics like Nielsen heuristics, but a managers can define their own custom set of heuristic sets by clicking on the create a new heuristic set tab in the sidebar. A page similar to the the following screenshot will opens and after entering a name and a description the set will be created after submitting and a page similar to the following screenshot will be opened. The set still does not have any principle and therefore is empty.




Adding new principles is done by clicking Add New Heuristic Principle button:





After adding some principles the set will be similar to the following screenshot. Each principle can be edited or be deleted using operation buttons in the last columns.




NOTE:

A set of heuristics which is in use in a project cannot be deleted as long as that project is using the set. To delete this set the manager can edit that project and chose another set of heuristics or even delete the whole project to make the set free to delete.



Defining a Project


Defining a new project is done by clicking on the Creare a New Project link in the sidebar of the dashboard to see a form like the following screenshot. Except the filed link, the other fields are required to be filled. After filling the form clicking submit will create the new project.




Merging Evaluations


Manager can merge existing evaluations and make merged evaluation. The merging can be done anytime but it is better to be after the deadline of the project. After clicking on the Merge Evaluations button in the project detail or in the overview of the project in the dashboard merge desktop opens a similar page to the following screenshot. In the first column with the name Evaluations (not merged) you can see all the evaluations which are not already merged, categorized by the name of their evaluators.





By clicking on each evaluation you can see the details of that evaluation. Getting similar evaluations is done after clicking the checkbox of any evaluation. Each time a checkbox is checked place-based and content-based recommendation for merge appears in the middle columns:




By choosing more than two checkboxes a blue button with the name Merge Selected Evaluations appears on the top left of the page:





Clicking this button will combines the information of all the selected evaluations to create a new merged evaluation. A form-page for evaluation with the information of the new merged evaluation will be opened which can/should be edited and the combined data in its fields be summerized.



Exporing a Report


In project detail page a manger can find different Export buttons which gives different export options in a dropdown menu to export both merged evaluations and normal evaluations separately.

The same as exporting evaluations in project detail page and also on the top of the merge desktop page the manager can find Export button for producing and exporting a report in different format.
